FoodGood Client Front

# Setup Web

## With NVM

Install [NVM](https://github.com/nvm-sh/nvm)

```shell
nvm install
nvm use
```

## Without NVM

Install the same [Node](https://nodejs.org/en) version of the `.nvmrc` file.

## Dependencies

```shell
npm ci
```

## Launch project

```shell
npm run start
```

## Launch test

```shell
npm run test
```

## Launch JSON Server

```shell
npm run start:server
```

# Story Book

```shell
npm run storybook
```
